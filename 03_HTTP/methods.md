Что такое HTTP-метод

HTTP-метод является частью запроса.

Он указывает серверу, какое действие клиент хочет выполнить над данными.

Основные HTTP-методы:

GET — получить данные.

Пример:
GET /cats

Клиент просит сервер предоставить список котиков.


POST — создать новую сущность.

Пример:
POST /cats

Клиент просит сервер создать нового котика.


PUT/PATCH — изменить существующую сущность.

Пример:
PATCH /cats/1

Клиент просит сервер изменить данные котика №1.


DELETE — удалить сущность.

Пример:
DELETE /cats/1

Клиент просит сервер удалить котика №1.
