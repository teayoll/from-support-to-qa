GET — получить данные
GET /cats
POST — создать новую сущность
POST /cats
PUT/PATCH — изменить существующую сущность
PUT /cats
DELETE — удалить сущность
DELETE /cats/1

HTTP-метод
является частью запроса
указывает на то, что именно хочет получить клиент от сервера (что хочет, чтобы сервер сделал) 

по одному примеру с котиками для каждого.
