
# JSON

JSON (JavaScript Object Notation) — популярный формат хранения и передачи данных.

Часто используется в API для передачи данных между клиентом и сервером.

Основные элементы:

Объект:
{
  "name": "Барсик",
  "age": 3
}

Массив:
[
  {
    "name": "Барсик"
  },
  {
    "name": "Мурка"
  }
]

Ключ — поле объекта.
Пример: "name"

Значение — данные поля.
Пример: "Барсик"

Важно:
Валидный JSON и корректные данные — не одно и то же.

JSON может быть валидным по синтаксису, но содержать неверные или неполные данные.
