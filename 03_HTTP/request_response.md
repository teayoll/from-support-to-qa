request - буквально про запрос 
клиент что-то запрашиввет (направляет сообщение) серверу

response - ответ
сервер что-то направляет клиенту

запрос состоит из:
- метод (например, GET) 
- адрес (например, /products)
- дополнительная информация
- тело запроса (не всегда)

ответ состоит из:
- код ответа (200, 404,500 или другой)
- заголовки
- тело ответа
